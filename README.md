# Telegram Discord VIP Bot V5

Sistema VIP BORDO CAMPO con pagamento Telegram Stars, ruolo Discord automatico, notifiche staff, DM utenti, rimozione ruolo alla scadenza e comandi staff.

## Installazione

```bash
npm install
copy .env.example .env
npm start
```

## Pubblicare i comandi Discord

```bash
npm run publish
```

## Variabili principali

```env
VIP_ROLE_ID=1518270281028997170
VIP_LOG_CHANNEL_ID=1518278744719102104
VIP_PANEL_CHANNEL_ID=1518278904475812020
VIP_STARS_PRICE=50
VIP_DURATION_DAYS=30
```
